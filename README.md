# 📝 To-Do List App

A clean and responsive To-Do List App built using **HTML**, **CSS**, and **JavaScript**.

This app allows users to:

✅ Add tasks  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Filter by All / Active / Completed  
✅ Edit tasks  
✅ Toggle Dark Mode  
✅ Persist data using LocalStorage
